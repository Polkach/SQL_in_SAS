Задание No2

В наборе данных ecsql1.salary_history содержится информация о
фактах изменения зарплат сотрудников компании.
Набор данных содержит следующие переменные: Employee_ID,
Employee_Gender, Salary_Change_Date и Salary - соответственно,
номер сотрудника, пол, дата установления зарплаты и уровень
зарплаты (действующий до следующего изменения зарплаты или
увольнении сотрудника). Т.е., например, если зарплата менялась 1
раз в год, то будет только 1 запись с датой Start_Date в этом году;
если зарплата менялась для конкретного сотрудника 5 раз за все
время его работы в компании, то будет 5 записей.

1) Напишите SQL-запрос, который для заданной даты (входной
параметр) выдаст зарплаты сотрудников по состоянию на указанную
дату.

2) Напишите программу (с использованием только процедуры
PROC SQL), которая для 2-х заданных дат (входные параметры)
выдаст информацию о среднем изменении зарплаты (%) между
указанными датами для мужчин и для женщин.
Пример отчета:

Employee_Gender	Date1		Date2		Change
F		01/07/2004	01/07/2005	4.00%
M		01/07/2004	01/07/2005	2.97%

3) Сколько времени работает программа из п.2?
(более точную информацию о времени работы каждого шага /
каждого оператора процедуры SQL дает глобальная опция fullstimer
/ опция процедуры SQL stimer)

4) Перепишите программу из п.2 с использованием DATA Step так,
чтобы по набору данных ecsql1.salary_history был только 1 проход.

5) Сколько времени работает программа из п.4?
